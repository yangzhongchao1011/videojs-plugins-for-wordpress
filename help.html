在WordPress编辑器中使用 插入HTML5视频 按钮 或者 [videojs] 短代码在你的页面或文章中使用以下设置:

mp4

对于 h.264/MP4 格式视频资源的位置

[videojs mp4="http://video-js.zencoder.com/oceans-clip.mp4"]

ogg

对于 Theora/Ogg 格式视频资源的位置

[videojs ogg="http://video-js.zencoder.com/oceans-clip.ogg"]

webm

对于 VP8/WebM 格式视频资源的位置

[videojs webm="http://video-js.zencoder.com/oceans-clip.webm"]

youtube

youtube资源链接

[videojs youtube="http://www.youtube.com/watch?v=DJU6ppZAaec"]

poster

视频封面图片的位置

[videojs poster="http://video-js.zencoder.com/oceans-clip.png"]

width

视频的宽

[videojs width="640"]

height

视频的高

[videojs height="264"]

preload

用户点击播放之前尽快加载视频。使用'auto'，'matadata'或'none'，在浏览器或设备允许时，元数据将是预加载视频的唯一的数据

[videojs preload="auto"]

autoplay

当视频准备好时播放视频，使用'true'或'false' （其实就是自动播放...）

[videojs autoplay="true"]

loop

是否设置视频为循环播放，使用'true'或'false'

[videojs loop="true"]

controls

使用'false'隐藏播放器控制按钮

[videojs controls="false"]

id

为你的播放器添加自定义ID

[videojs id="movie-id"]

class

为你的视频添加自定义Class,充分利用悬浮使你的视频 '左对齐' 或 '右对齐'

[videojs class="alignright"]

Tracks

文本轨道是为HTML5视频出发文本查看器的时间函数，把[track]短码放入[videojs]的短码中就能使用轨道，你可以为kind,src,srclang,label,default属性赋值，更多信息可在Video.js Documentation.

[videojs][track kind="captions" src="http://video-js.zencoder.com/oceans-captions.vtt" srclang="en" label="English" default="true"][/videojs]
